第1天（5.22）#

接机

晚餐：名悦餐厅 或 荣记 或 江南晓菜 或 江南里 或 蟹叁宝 或 沈灶 或 菌味斑鱼府 或 广东阿生鱼捞

买糕点 泸溪河

酒店：怡莱酒店

第2天（5.23)#

早餐：鸭得堡鸭血粉丝、除此以外还有安庆馄饨、螺鼎记螺狮粉、鸡鸣汤包等

美龄宫 预计2 小时

中餐：自带食物 糕点、酸奶等

观光车

中山陵 预计2 小时

观光车

明孝陵 预计 2 小时 或者 植物园

晚餐：南京大牌档 或 菌味斑鱼府 或 江南里 或 蟹叁宝 或 沈灶 或 广东阿生鱼捞

休息

酒店：派柏 云酒店 570

第3天（5.24)#

早餐：汪家馄饨

甘家大院 预计2小时

中餐：陇味府满 牛肉面

瞻园 2-3 小时

夫子庙 2-3 小时

乌衣巷

晚餐：老门东 德州扒鸡-藕园 小郑酥烧饼 一籴酒酿-桂花酒酿小圆子 豆腐王朝-暴雪臭豆腐 庙东排骨-掌中宝 沈记臭豆腐

休息

酒店：派柏 云酒店

第4天（5.25)#

早餐：老太皮肚面

休息 或 逛江南科举博物馆

中餐：川味1号怀旧火锅 或 无界釜（那还可以先逛一下江南科举博物馆-江南贡院）

雨花台 南门进

老门东 或 先游秦淮河

晚餐：老门东 陆氏梅花糕 司记豆腐脑 鸡鸣汤包 鸭来哒-烤鸭肠 咸亨传统小吃-大鱿鱼 鸡汁回卤干

秦淮河：

  1. 日景 抖音可能会便宜
  2. 划船 预计 1 小时 上船面向船尾右手边最头和最尾的位置 可以二维码购票 18：15-20 西段
  3. 步行 东园桥、文德桥、平江桥

酒店：派柏 云酒店

第5天（5.26)#

早餐：李记清真馆 牛肉锅贴 牛杂粉丝

鸡鸣寺 2-3 小时

中餐：小厨娘

明城墙 解放门 -> 玄武门

玄武湖 回到解放门

晚餐：科巷 三姐酒酿 小郑酥烧饼 德州扒鸡藕圆 许阿姨青团 鸡蛋汉堡 天津童子鸡 洪福面馆

按个摩？

酒店：骏怡连锁酒店 或 橙悦酒店

第6天（5.27)#

要不要直接收了东西带着去玩，买泸溪河糕点

早餐：蒸饭

总统府 预计2-3小时 提前一天 公众号 南京总统府预约 预计8：30 - 11：00结束

中餐：卓粤拉肠 或者 好婆汤包

去机场 预计12:30 出发，14:00 左右到
